# 📝 To-Do List

A clean and responsive To-Do List web application built using **HTML, CSS, and JavaScript**. It helps users organize daily tasks with an intuitive interface and a smooth user experience.



## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- ✏️ Edit existing tasks
- 📱 Responsive design for mobile and desktop
- 💾 Stores tasks using Local Storage
- 🎨 Modern and user-friendly interface
- ⚡ Fast and lightweight



## 🛠️ Built With

- HTML5
- CSS3
- JavaScript 
- Local Storage API
